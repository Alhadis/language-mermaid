Mermaid language support for Atom
=================================

Syntax highlighting for [Mermaid][1] diagrams.

Currently in-development; intended to address [`github/linguist#5578`][2].

Supported graph types
---------------------
* [x] Flowchart
* [ ] Sequence diagram
* [ ] Class diagram
* [ ] State diagram
* [ ] Entity relationship diagram
* [ ] User journey
* [ ] Gantt
* [ ] Pie chart
* [ ] Requirement diagram
* [ ] Gitgraph
* [ ] C4C diagram
